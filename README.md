# Login_system

Система позволяющая создавать пользователей с определенными ролями и определенным доступом к ресурсам. С помощью модели AccessRoleRule происходит связывание роли и ресурса, а с помощью UserRole связывание пользователя с ролью. В файле permissions.py проверяется наличие доступа к ресурсу определенной ролью.
